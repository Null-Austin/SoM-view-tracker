Overview

This is a view counter that makes SVG badges. When someone loads the image, it counts the view and shows the numbers.

How it works

Put the SVG image in your README or website. Each time it loads, the counter goes up and shows the new number.

Badge styles

  • default.svg - Simple text
  • dark.svg - Styled box

Usage

h1.text-2xl > a:nth-child(1) > span:nth-child(1)::after {
content: "";
display: inline-block;
width: 3em;
height: 1em;
background: url('https://eleven-deficit-formats-white.trycloudflare.com/svg/views/18445/dark.svg') no-repeat center center;
background-size: contain;
vertical-align: middle;
}

How it counts

View tracking

  • Unique views: One per IP address
  • Total views: Every time someone loads it
  • Privacy: IP addresses are hashed
  • Storage: SQLite database file

No caching

The images don't get cached so the counts stay accurate.
